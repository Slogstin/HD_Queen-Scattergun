# "Queen" Scattergun

This Addon adds .066 Bore Shotgun to [Hideous Destructor](https://codeberg.org/mc776/HideousDestructor).

Requires [HDCoreLib](https://github.com/HDest-Community/hdest-core-lib).

## Contents

### ''Queen'' Scattergun

    * Can spawn on Shotguns
    * Uses .066 Shells
    * Might jam sometimes
    * Fits in a backpack
    * Can be sold by Merchants
    * Price and Spawn weight can be adjusted in the menu
    * Loadout code: QSG
    * Class: VirQueen

### .066 Scattergun Magazine

    * Can spawn on Shells
    * Holds 16 shells
    * Can be sold by Merchants
    * Price and Spawn weight can be adjusted in the menu
    * Loadout code: QNM
    * Class: VirQueenMag
